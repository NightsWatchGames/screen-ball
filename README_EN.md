# screen-ball 
work and play ball
- [x] Ball simulation
- [x] Mouse control
- [x] Load 3d model
- [x] Pass through cursor events
- [ ] Multiple monitors support（Blocking by[bevyengine/bevy#7600](https://github.com/bevyengine/bevy/issues/7600) 和 [bevyengine/bevy#7975](https://github.com/bevyengine/bevy/issues/7975)）

## Get started
1. Native
```
cargo run
```
