# screen-ball 屏幕踢球游戏
- [x] 足球运动模拟
- [x] 鼠标控制踢球
- [x] 3d模型载入
- [x] 鼠标输入等事件落入下面窗口
- [ ] 多显示器支持（卡在[bevyengine/bevy#7600](https://github.com/bevyengine/bevy/issues/7600) 和 [bevyengine/bevy#7975](https://github.com/bevyengine/bevy/issues/7975)）

## 运行
1. 本地运行
```
cargo run
```
